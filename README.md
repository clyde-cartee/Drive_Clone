# Drive_Clone

This is a google drive clone

## Start up
Connect to hopper, run app, login to daytona is within the file, go to http://localhost:8000

## Registering/Log in
When opening up the site (localhost:8000), first pick register. Add your username and password then you will be redirected to log in where you can use that 
username and password to log in.

## upload file
click drop a file or click to upload (will be updated later to allow drag and drop), then press upload. 

## Add A Folder

click the new folder bar and add a folder name. press the +folder buton to add this folder to current location. 

## file/folder movement
You can drag and drop files or folders into other folders. You can do this by using the my folders sidebar to view folders and their nested folders. 
You can also use the choose folder and move button to move. 

## share and rename. 
Right clicking on file or folder allows the options rename and share. (Currently folders cannot be shared.)
Sharing can be done either with username of another person or creating a public link that allows anyone to view. 
Rename is straight forward with writing name and entering to change it. 

## search bar
You can search for folders or files in that order of priority if file and folder share a name

## view/ download/ delete
You can view download or delete a file with the buttons present on the file. Folders have an X underneith it to delete folders. The files inside will just float to the root.

## folder navigation
You can navigate folders by clicking the folder, sidebar page with folders or home for root, or with bread crum links that appear when you enter a nested file.

## logout
press log out button to return to login page
